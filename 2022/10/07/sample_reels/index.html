# 声明

我个人只是将绘画当做一种娱乐的爱好，没有经历过任何专业的学习，所以在技术上存在很多欠缺，还请诸位见谅。虽然画的不多（所有的存稿都展示在这里了），但我还是非常喜欢画画的，往后在时间和精力都允许的情况下，一定增进自身技术的。最后祝大家玩的开心！

  • 注意：以下所有图片在时间上最近的都会展示在顶部

# 褶皱练习（二次加工）

  • 暑假当时浏览了国内外好多画师的作品，他们当中很多人的褶皱处理都非常厉害，于是抱着好奇的心态试了一下

# 闪光亚丝娜

  • 第一次学会了铅笔的涂色控制，打从心底的高兴

# 机甲之随随便便（原创）

  • 画的时候，清楚的记得当时下着大雨，莫名感到低落

# 战损机甲（原创）

  • 大一刚进入画社时，心血来潮画了一张

# 简笔坦克（原创）

  • 高中时候画的，嘿嘿，没想到还能侥幸翻找到，不过也就幸存这一张了。这张图高二下学期画了一半，余下的一半在高三下学期完成
